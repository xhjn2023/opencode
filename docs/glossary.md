# 记账应用 - 术语表 (Glossary)

## 核心概念

| 术语 | 英文 | 定义 |
|------|------|------|
| 账目 / 交易 | Transaction | 每一笔收入或支出的记录，包含金额、分类、日期、备注等信息 |
| 收入 | Income | 资金流入，如工资、兼职、投资收益等 |
| 支出 | Expense | 资金流出，如餐饮、交通、购物等 |
| 分类 | Category | 对收支的归类，如"餐饮"、"交通"、"工资"等，每种分类可设定类型（收入/支出） |
| 预算 | Budget | 对某个月份的某个分类（或全局）设定的支出限额 |
| 账单列表 | Ledger | 按时间顺序排列的所有交易记录列表 |
| 仪表盘 | Dashboard | 概览页面，展示总收支、余额、近期趋势等关键数据 |

## 界面元素

| 术语 | 定义 |
|------|------|
| 记账表单 | 用于录入一笔新交易的输入表单 |
| 分类管理 | 管理收支分类的 CRUD 界面 |
| 统计图表 | 以图表形式展示收支趋势和分类占比 |
| 搜索筛选 | 按日期范围、分类、类型、金额等条件过滤交易记录 |
| 预算看板 | 显示当月各分类预算执行情况的界面 |

## 数据字段

| 字段 | 说明 |
|------|------|
| 金额 (amount) | 交易的数值，正数表示收入，支出存储为正值由 type 字段区分 |
| 类型 (type) | "income" 或 "expense" |
| 日期 (date) | 交易发生的日期 (YYYY-MM-DD) |
| 备注 (note) | 用户对这笔交易的自定义说明 |
| 月份 (month) | 预算所属月份 (YYYY-MM) |
